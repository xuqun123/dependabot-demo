# Security Policy

## About This Repository

This is a demo repository for testing Dependabot features. It intentionally contains outdated dependencies for demonstration purposes.

## Supported Versions

As a demo repository, there are no production releases. Security updates are applied on a best-effort basis to keep the demo functional.

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it by:

1. **Opening a GitHub issue** in this repository describing the vulnerability.
2. **Emailing the maintainer** with details of the vulnerability.

Since this is a demo repository, please note that intentionally outdated dependencies are expected and do not need to be reported.

## Security Update Policy

- Dependabot is enabled to automatically detect and propose updates for known vulnerabilities.
- Security patches are reviewed and merged periodically.
- This repository is not intended for production use.
